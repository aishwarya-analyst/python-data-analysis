# python-data-analysis
Jupyter Notebook assignments covering core Python 
data analysis libraries.

## 📚 Notebooks

| Notebook | Topics Covered |
|----------|---------------|
| numpy.ipynb | Arrays, slicing, math ops, vectorization |
| pandas.ipynb | DataFrames, filtering, groupby, merge, nulls |
| matplotlib.ipynb | Line, bar, histogram, pie charts |
| seaborn.ipynb | Scatter, box, count, bar plots |

## 🛠️ Libraries Used
Python, NumPy, Pandas, Matplotlib, Seaborn
